## Featured Project

### [Multi-Asset Portfolio Strategy](https://github.com/amritshakya/multi-asset-portfolio-strategy)

Designed and backtested a constrained multi-asset portfolio across U.S. equities, emerging markets, energy, gold, and Bitcoin.

Evaluated asset-allocation tradeoffs, risk contributions, drawdowns, benchmark-relative performance, and sensitivity across macroeconomic regimes.

**Methods:** Mean-variance optimization, Monte Carlo simulation, rolling analysis, portfolio risk measurement
**Tools:** R, Excel
